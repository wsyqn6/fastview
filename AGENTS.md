# FastView Agent Configuration

## 项目定位
**专注图片查看**的轻量工具，追求：
- 🚀 **启动快**：< 1秒
- 💾 **体积小**：~5MB（发布版）
- 🖼️ **格式全**：JPEG/PNG/GIF/WebP/BMP/TIFF/ICO/AVIF
- ✨ **UI现代**：简洁流畅的即时界面

## 技术选型
**原则**：最小依赖、零运行时、原生性能

| 组件 | 选型 | 理由 |
|------|------|------|
| GUI | Eframe (egui) 0.32 | 即时模式，无Webview，体积小 |
| 图片解码 | image-rs 0.25 | 纯Rust，支持主流格式 |
| 文件对话框 | rfd 0.16 | 调用系统原生对话框 |
| 配置 | serde + dirs | 最小化依赖 |

## 项目结构
```
src/
├── main.rs    # 入口（模块声明 + 启动）
├── types.rs   # 类型定义
├── fonts.rs   # 字体加载
└── app.rs     # 应用逻辑
```

## UI 设计规范

### 设计原则
- **轻量简洁**：避免过度装饰，符合"体积小、启动快"定位
- **视觉一致**：统一设计语言，使用 egui 主题色变量适配深浅主题
- **即时响应**：无动画过渡，点击即响应

### 核心样式
- **毛玻璃效果**：`panel_fill.gamma_multiply(0.7-0.85)`
- **边框/阴影**：透明度 12-30%
- **圆角**：小 4px / 中 6px / 大 8-10px
- **间距**：小 4px / 中 8px / 大 12-14px

### 字体规范
- **正文字体**：系统默认（Windows: 微软雅黑）
- **等宽字体**：`FontFamily::Monospace`（技术信息）
- **字号层级**：标题 14-16px / 正文 12px / 次要 10px / 快捷键 11px
- **重要**：所有字体大小使用整数值

### 组件要点
- **状态栏**：底部居中悬浮，自适应宽度，最大 800px
- **菜单栏**：顶部固定 24px，四个主菜单
- **弹窗**：居中对齐，不可调整大小，固定尺寸
- **全屏模式**：隐藏菜单栏和状态栏

### 国际化
- 所有文本通过 `TextKey` 枚举管理，禁止硬编码
- 新增功能时同步添加中英文翻译

## 注意事项
- Windows 自动加载微软雅黑，其他系统尝试 Noto Sans CJK
- 配置位置：`%APPDATA%\fastview\settings.json` (Windows)
- 默认缓存 10 张图片，可在设置中调整
